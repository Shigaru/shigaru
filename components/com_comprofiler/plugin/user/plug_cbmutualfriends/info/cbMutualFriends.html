CB Mutual Friends Plugin

This page documents the CB Mutual Friends Plugin (AXXIS Internet Solutions). This Community Builder plugin provides profiles with a Mutual Friends tab.

How to install the plugin?

All community builder plugins are installed via the community builder plugin management backend panel.

How to configure the plugin?

There are currently two (2) configuration parameters that can be set for the plugin. These are available on the tab's page.

How to use the plugin

Once the relevant parameters are set by the admin in the plugin and tab area in the backend, users should be able to access their very own Mutual Friends tab in the frontend. While seeing any other user profile, their common friends will be displayed under this tab.
